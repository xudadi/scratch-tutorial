# 事件

事件指令是指令的启动器，作为指令串的第一个指令使用，前面不能拼接其他积木。当某个事件的条件被满足时，事件积木开始被执行。

# 角色积木介绍

# 当绿旗被点击

功能：当舞台上面的绿旗按钮被点击时，该积木开始被执行。连续触发时，会立即重新执行指令串。

参数：无

返回值：无

难度：☆☆

讲解视频open in new window

# 当按下[空格]键

功能：当按下键盘上某个按键时，该积木开始被执行。连续触发时，需等待已经在执行的指令串执行完毕后，才能重新触发生效。

参数：固定参数积木，选项为空格、上下左右方向键、任意、a-z、0-9。

空格、上下左右方向、a-z、0-9键，按下所选择的按键时，开始执行该积木。

任意，按下任意按键时，开始执行该积木。

返回值：无

难度：☆☆

讲解视频open in new window

# 当角色被点击

功能：当舞台中的角色被点击时，该积木开始被执行。连续触发时，会立即重新执行指令串。

参数：无。

返回值：无

难度：☆☆

讲解视频open in new window

# 当背景换成[背景1]

功能：当背景切换成所选择的背景时，该积木开始被执行。连续触发时，会立即重新执行指令串。

参数：固定参数积木，选项为作品中的背景名称。

返回值：无

难度：☆☆

讲解视频open in new window

# 当[响度]＞(10)

功能：当麦克风接收的响度大于设置的数值，或计时器大于设置的数值时，该积木开始被执行。连续触发时，需等待已经在执行的指令串执行完毕后，才能重新触发生效。

类型参数：固定参数积木，选项为响度、计时器。

数值参数：椭圆形参数，进行比较的数值。

返回值：无

难度：☆☆

讲解视频open in new window

# 当接收到[消息1]

功能：当接收到对应的广播消息时，该积木开始被执行。

可以接收来自同一角色的消息，也可以接收来自不同角色的消息。

接收到消息时，所有该消息事件积木会同时执行。

连续收到消息时，会立即重新执行指令串。

参数：固定参数积木，选项为广播的消息。

返回值：无

难度：☆☆☆

讲解视频open in new window

# 广播[消息1]

功能：广播一条消息，通知接收消息事件开始执行。广播消息后继续执行该积木后的指令。

参数：椭圆形下拉参数，选项为自定义的消息。

先自定义消息内容，在选中进行广播。

返回值：无

难度：☆☆☆

讲解视频open in new window

# 广播[消息1]并等待

功能：广播一条消息，通知接收消息事件开始执行。广播消息后停止执行该积木后的指令，等待接收消息事件所在指令串执行完成后，继续执行该积木后的指令。

参数：椭圆形下拉参数，选项为自定义的消息。

先自定义消息内容，在选中进行广播。

返回值：无

难度：☆☆☆

# 并行事件

多个相同事件积木会被同时触发，并行执行多个指令串。利用并行事件可以让角色同时完成多个操作。

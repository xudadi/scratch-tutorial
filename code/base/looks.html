# 外观

外观是什么？

外观是角色在舞台上展现出来的样子，例如它使用的造型、大小、颜色等。

外观指令用于调整角色和背景在舞台中的显示效果，包括说话、造型、大小、特效等。

前置知识：背景区，造型（背景），大小，显示，角色的层级

# 换成([造型1])造型

功能：根据参数所选造型名称，切换为相应造型。

参数：椭圆形下拉参数，默认选项有角色的造型名称。添加造型后，选项增加造型名称。

可以使用变量作为参数，变量值设置为造型名称或编号。

当变量值设置的数字大于编号值时，程序会从头开始计算。例如，假设一个角色有3个造型，造型编号分别为1,2,3。变量值为4时，将切换为造型1.变量值为5时，切换为造型2，以此类推。

当变量值为0时，将切换为最后一个造型。变量值继续减少为负数时，将从最后一个开始，反向切换。例如，假设一个角色有3个造型，造型编号分别为1,2,3。变量值为-1时，切换为倒数第2个造型。变量值为-2时，切换为倒数第3个造型，以此类推。

返回值：无

难度：☆☆

# 换成([背景1])背景

功能：切换舞台背景。

参数：椭圆形下拉参数，默认选项有背景名称、下一个背景、上一个背景和随机背景。添加背景后，选项增加背景名称。

背景名称为当前舞台拥有的背景名称，选择后将背景切换为对应的背景图片。可以使用变量作为参数，变量值设置为背景名称或编号。

下一个背景，按照背景编号顺序，从小到大，依次切换背景图片。

上一个背景，按照背景编号顺序，从大到小，依次切换背景图片。

随机背景，自动切换为任意一个背景图片。

返回值：无

难度：☆☆

参数大于0，角色变大。参数小于0，角色变小。角色的大小会受到当前造型的限制。

# 将大小设为(100)

功能：设置角色的大小。

角色的大小会受到当前造型的限制。例如将默认的小猫角色设置为造型1，此时该角色大小最大为540，最小为5。角色ball的大小最大为1174，最小为11。可以给小猫添加ball-a造型，通过在造型1和ball-a之间进行切换，从而突破原本造型大小的限制。

参数：椭圆形参数。

返回值：无

难度：☆

# 将[颜色]特效增加(25)

功能：增加或减少角色的特效值，默认值为0。多个特效可以组合使用。

特效类型参数：固定参数积木，选项为颜色、鱼眼、漩涡、像素化、马赛克、亮度和虚像。

颜色特效，改变角色的颜色，将角色变为粉红色、绿色、蓝色、紫色或中间的任意颜色。设置数值越大，变化越大，取值范围0~200。

鱼眼特效，设置数值＞0，角色从中部开始扩展。数值＜0，角色从中部开始收缩。设置数值越大，效果越明显。

漩涡特效，设置数值＞0，角色从中部开始逆时针旋转。数值＜0，角色从中部开始顺时针旋转。设置数值越大，效果越明显。

像素化特效，角色像素化，设置数值越大，角色越模糊。

马赛克特效，角色分裂成多个相同的小角色，设置数值越大，角色分裂越多。

亮度特效，设置数值＞0，角色变亮，数值＜0，角色变暗。设置数值越大，效果越明显。取值范围-100~100。

虚像特效，角色变透明，设置数值越大，效果越明显。取值范围0~100。角色使用的说话和思考气泡不受影响。

数值参数：椭圆形参数，增加的特效数值。

返回值：无

难度：☆☆

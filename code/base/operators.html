# 角色积木介绍

# 加法

功能：计算两个加数的和。用于椭圆形参数积木和椭圆形下拉参数积木中。

参数：椭圆形参数，2个需要相加的数字。

返回值：两个加数的和。

参数：椭圆形参数，2个需要相减的数字。

返回值：两个数的差。

# 乘法

功能：计算两个数的乘积。用于椭圆形参数积木和椭圆形下拉参数积木中。

参数：椭圆形参数，2个需要相乘的数字。

返回值：两个数的乘积。

难度：☆☆

讲解视频open in new window

# 除法

功能：计算两个数的商。用于椭圆形参数积木和椭圆形下拉参数积木中。

参数：椭圆形参数，2个需要相除的数字。

一个正数除以0时，返回Infinity，正无穷大。一个负数除以0时，返回-Infinity，负无穷大。0除以0时，返回NaNopen in new window（Not a Number），表示不是数字。

返回值：两个数的商。

难度：☆☆

讲解视频open in new window

# 在(1)和(10)之间取随机数

功能：返回两个参数之间的随机数open in new window，返回的随机数范围包含填入的参数。用于椭圆形参数积木和椭圆形下拉参数积木中。

参数：椭圆形参数，随机数的取值范围。

当2个参数为整数时，返回随机整数。

当2个参数中任意一个为小数时，返回随机小数。

返回值：2个参数之间的随机数。

难度：☆☆

讲解视频open in new window

# 大于

功能：比较两个字符的大小，第一个参数大于第二个参数，返回true，第一个参数小于或等于第二个参数，返回false。用于菱形参数积木中。

参数：椭圆形参数，两个用于比较的字符。

参数为数字时，按数字大小进行比较。

参数为字符时，使用字符对应的Ascii码open in new window进行比较，英文字母不区分大小写。

返回值：true或false。

难度：☆☆

讲解视频open in new window

# 小于

功能：比较两个字符的大小，第一个参数小于第二个参数，返回true，第一个参数大于或等于第二个参数，返回false。用于菱形参数积木中。

参数：椭圆形参数，两个用于比较的字符。

参数为数字时，按数字大小进行比较。

参数为字符时，使用字符对应的Ascii码open in new window进行比较，英文字母不区分大小写。

返回值：true或false。

难度：☆☆

讲解视频open in new window

# 等于

功能：比较两个字符的大小，第一个参数等于第二个参数，返回true，第一个参数不等于第二个参数，返回false。用于菱形参数积木中。

参数：椭圆形参数，两个用于比较的字符。

参数为数字时，按数字大小进行比较。

参数为字符时，使用字符对应的Ascii码open in new window进行比较，英文字母不区分大小写。用来判断两个文字是否相同。

返回值：true或false。

难度：☆☆

讲解视频open in new window

# 与

功能：用来判断两个条件是否同时成立，两个参数都为true时，返回true，两个参数中有任意一个为false时，返回false。用于菱形参数积木中。

参数：菱形参数积木，需要比较的两个条件。

true 与 true = true

true 与 false = false

false 与 true = false

false 与 false = false

返回值：true或false。

难度：☆☆

讲解视频open in new window

# 或

功能：用来判断两个条件是否有任意一个成立，两个参数中有任意一个为true时，返回true，两个参数都为false时，返回false。用于菱形参数积木中。

参数：菱形参数积木，需要比较的两个条件。

true 与 true = true

true 与 false = true

false 与 true = true

false 与 false = false

返回值：true或false。

难度：☆☆

讲解视频open in new window

# 不成立

功能：用来获取与参数条件相反的结果。参数为true时，返回false，参数为false时，返回true。用于菱形参数积木中。

参数：菱形参数积木，初始条件。

true 不成立 = false

false 不成立 = true

返回值：true或false。

难度：☆☆

讲解视频open in new window

# 连接(苹果 )和(香蕉)

参数：椭圆形参数，需要拼接的两个字符串。

返回值：拼接后的字符串

难度：☆☆

讲解视频open in new window

# (苹果)的第(1)个字符

功能：获取参数字符串中的第几个字符。用于椭圆形参数积木和椭圆形下拉参数积木中。

字符参数：椭圆形参数，字符串。

位置参数：椭圆形参数，需要获取的位数。

位数从1开始，代表左侧第一个字符，依次递增。

位数超过字符长度时，返回空。

返回值：字符

难度：☆☆

原始字符参数：椭圆形参数，原始字符串。

# 角色积木介绍

# 等待(1)秒

功能：当指令串执行到该积木时停止执行，等待设置的时间后，再继续执行后面的指令。

参数：椭圆形参数，需要停止的秒数。

返回值：无

难度：☆

讲解视频open in new window

# 重复执行(10)次

功能：重复执行放入该积木中的指令串，重复的次数为设置的数值。

参数：椭圆形参数，需要重复执行的次数。

返回值：无

难度：☆☆

讲解视频open in new window

# 重复执行

功能：无限次重复执行放入该积木中的指令串，可使用停止按钮或停止[全部脚本]结束循环。

参数：无

返回值：无

难度：☆☆

讲解视频open in new window

# 如果<>那么

功能：当参数返回true时，执行该积木中的指令串。

参数：菱形参数积木，需要满足的条件。

返回值：无

参数：菱形参数积木，需要满足的条件。

返回值：无

参数：菱形参数积木，需要满足的条件。

返回值：无

难度：☆☆☆

讲解视频open in new window

# 停止[全部脚本]

功能：停止正在运行的指令串。

参数：固定参数积木，选项为全部脚本、这个脚本和该角色的其他脚本。

全部脚本，停止作品中的所有指令串，等同于点击停止按钮。

这个脚本，停止该积木所在的指令串。

该角色的其他脚本，停止当前角色中，该积木不在的其他指令串。

返回值：无

难度：☆☆☆

讲解视频open in new window

# 当作为克隆体启动时

功能：当一个角色被克隆时，该积木开始被执行，用于控制克隆体的操作。

参数：无

返回值：无

难度：☆☆☆

讲解视频open in new window

# 克隆[自己]

功能：克隆一个选择的角色，生成的克隆体与被克隆角色属性（坐标、方向、大小、颜色、隐藏状态）一致。克隆后，通过当作为克隆体启动时积木对克隆体进行控制。

一个作品最多只能有300个克隆体，克隆体越多，作品运行越卡顿，所以当一个克隆体不再需要时，应使用删除此克隆体将其删除。

参数：固定参数积木，选项为自己和其他角色名称。

返回值：无

难度：☆

讲解视频open in new window

# 删除此克隆体

功能：删除生成的克隆体，需要配合事件积木使用。

点击停止按钮或使用停止全部脚本，可以删除所有克隆体。

参数：无

返回值：无

难度：☆

讲解视频open in new window

# 克隆与事件

使用当按下空格键、当背景换成背景1、当响度＞10、当接收到消息1积木时，原始角色和所有克隆体都会执行该事件积木后的操作。

使用当角色被点击积木时，只有当前被点击角色或克隆体会执行该事件积木后的操作。

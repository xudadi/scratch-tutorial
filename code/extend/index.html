# 简介

Scratch在基础指令类型之外，还隐藏了许多扩展模块。

每种扩展模块都能完成特定的功能，例如演奏音乐、画笔、视频侦测和乐高机器人等。从功能上来说大致可以将它们分为两类，软件类模块和硬件类模块。

Scratch从3.0版本开始，将软件的开发语言换成了HTML5，并且完全开源。因此，只要你懂得HTML5开发，就可以制作属于自己的扩展模块。

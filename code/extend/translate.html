# 角色积木介绍

# 将(你好)译为[(爱沙尼亚语)]

功能：将参数设置的文字翻译为参数所选的语言。

文字参数：需要被翻译的文字。

语言参数：椭圆形下拉参数，翻译的目标语言。

返回值：翻译后的文字

难度：☆

演示视频open in new window

# 访客语言

功能：当前软件所设置的默认语言。作为变量使用。

参数：无

返回值：默认语言

难度：☆

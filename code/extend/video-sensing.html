# 视频侦测

视频侦测模块可以做出体感游戏的效果。它能利用电脑上的摄像头，捕捉画面中物体的移动状态，实现现实物体与程序角色的交互。

# 角色积木介绍

# 当视频运动＞(10)

功能：当视频中物体相对于角色的运动量大于设置的数值时，该积木开始被执行。参考相对于([角色])的视频([运动])

参数：椭圆形参数，进行比较的数值，运动量最大值为100。

返回值：无

难度：☆

演示视频open in new window

# 相对于([角色])的视频([运动])

功能：视频中物体，相对于角色或舞台的运动量或运动方向。

参考对象参数：椭圆形下拉参数，默认选项有角色和舞台。

角色，以当前角色为参考对象，计算视频中运动物体的运动量或运动方向。

舞台，以整个舞台为参考对象，计算视频中运动物体的运动量或运动方向。

计算类型参数：椭圆形下拉参数，默认选项有运动和方向。

运动，视频中物体相对参考对象的运动量，以数字表示。视频中移动的物体和幅度越大，返回的数字越大。视频中没有任何物体运动时，返回0。摄像头关闭时，返回-1。

相对于角色的运动量最大值为100。

方向，视频中物体相对参考对象的运动方向，以数字表示。该数字与角色的方向含义相同。

返回值：代表运动量和方向的数字

难度：☆☆☆

# [(开启)]摄像头

功能：开启、关闭摄像头，或将摄像头变为镜像模式。

参数：椭圆形下拉参数，默认选项有关闭、开启和镜像开启。

关闭，关闭摄像头。

开启，打开摄像头，舞台背景变为摄像头实时拍摄的画面。

镜像开启，打开摄像头，舞台背景变为摄像头实时拍摄的画面，画面中的内容与现实环境左右颠倒。

如果用手在摄像头前划过，舞台中物体左右移动的方向和真实的移动方向相反。

返回值：无

难度：☆

# 将视频的透明度设为(50)

功能：调整视频画面的透明度。

参数：椭圆形参数，需要设置的画面透明度。

数字为0，画面最清晰，数字越大，画面越透明，数字为100，画面全部消失。

返回值：无

难度：☆

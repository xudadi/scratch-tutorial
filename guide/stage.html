# 舞台区

舞台用于演示作品效果，拖动角色，更换角色的位置和层级。

# 绿旗和停止按钮

点击绿旗按钮，启动当绿旗被点击事件，开始运行作品。

点击停止按钮，停止作品运行。

# 三种视图模式

舞台拥有三种不同的展示方式，分别是迷你视图、默认视图和全屏展示，便于进行不同的操作。

# 迷你视图

舞台最小化，方便查看左侧的代码。

# 默认视图

进入软件时的默认展示方式。

# 全屏模式

舞台最大化，用于作品完成后的效果演示。

# 舞台

显示背景和角色，展示作品效果。

# 角色的层级

多个角色在舞台中重合时会发生遮挡，最后被拖动的角色会被移到最前面，层级为1。后面角色的层级依次递增。

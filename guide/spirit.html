# 角色区

角色是在舞台中表演的演员，每个角色拥有自己的属性和造型，再通过指令操控角色完成表演。

角色区用于展示、添加和删除角色，以及修改角色的属性。

# 4种添加角色的方式

选择一个角色

点击“选择一个角色”，可以从Scratch角色库中选择一个角色。选择一个角色时，鼠标停留在角色图片上，可以查看角色多造型切换的动画open in new window效果。

绘制

点击“绘制”，可以在绘制界面，使用画图工具画出一个角色造型。

随机

点击“随机”，Scratch会随机从角色库中选择一个角色。

上传角色

点击“上传角色”，可以从电脑中选择一张图片，上传到Scratch中作为角色使用。

上传一张图片代表一个角色，同一个角色的多个造型需要在造型区上传。

# 选中和删除角色

点击角色列表中的任意角色，可以将它选中。

点击选中角色右上角的“垃圾桶”图标，可以将它删除。

# 角色属性

展示角色的名称、坐标、显示状态、大小和方向。

可以手动修改，调整角色属性。

# 坐标

坐标代表角色在舞台中的位置，使用两个数字（x，y）表示。

x为横坐标，表示角色横向的位置，-240 ≤ x ≤ 240；向右移动，x增大，向左移动，x减小。

y为纵坐标，表示角色纵向的位置，180 ≥ y ≥ -180。向上移动，y增大，向下移动，y减小。

舞台中央的坐标为（x:0，y:0）。

角色通常在舞台上会覆盖一块区域，它的坐标值由区域中的一个点代表，也就是角色在造型绘制区中的坐标点决定。

# 显示

显示代表角色是否在舞台中显示，有显示和隐藏两种状态。

选择显示，角色会出现在舞台中；选择隐藏，角色会从舞台中消失。

# 大小

大小代表角色的体积，使用数字表示，数字越大，体积越大。

# 方向

方向代表角色的朝向，使用数字表示，0 ≤ 方向 ≤ 180为右半部分，-179 ≤ 方向 ≤ -1为左半部分。

0指向上，90指向右，180指向下，-90指向左。

调整方向时，角色会有3种不同的旋转模式。

任意旋转

选择任意旋转时，角色的朝向随着方向的改变而改变。

左右翻转

选择左右翻转时，0 ≤ 方向 ≤ 180，角色朝向右边。

-179 ≤ 方向 ≤ -1，角色朝向左边。

不旋转

选择不旋转时，角色的朝向保持固定，不会随着方向的改变而改变。

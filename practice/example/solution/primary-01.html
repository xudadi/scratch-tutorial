# 初级篇——你说我画

# 【疑惑篇】

在你开始使用Scratch编程之前，可以先找一个朋友玩一次“你说我画”的小游戏。每人拿一张白纸，按照下面的说明进行6个回合的游戏。

第一回合，先在两张白纸上分别画一条横线，接着分配游戏角色。一个人在横线上任意画一个圆，不能让另一个人看到。接着画圆的人需要通过说话的方式，让另一个人在相同的位置也画出一个圆。

第二回合，先在两张白纸上分别画一条横线，接着使用竖线将横线平均分为多格，分配游戏角色。一个人在任意一个竖线处画一个圆，不能让另一个人看到。接着画圆的人需要通过说话的方式，让另一个人在相同的位置也画出一个圆。

第三回合，先在两张白纸上分别画一条横线，接着使用竖线将横线平均分为多格，从0开始给每条竖线标出一个数字，分配游戏角色。一个人在任意一个竖线处画一个圆，不能让另一个人看到。接着画圆的人需要通过说话的方式，让另一个人在相同的位置也画出一个圆。

第四回合，先在两张白纸上分别画一个长方形，接着分配游戏角色。一个人在长方形中间的任意位置画一个圆，不能让另一个人看到。接着画圆的人需要通过说话的方式，让另一个人在相同的位置也画出一个圆。

第五回合，先在两张白纸上分别画一个长方形，接着将长和宽平均分为多格，分配游戏角色。一个人在长方形中任意一个交叉点画一个圆，不能让另一个人看到。接着画圆的人需要通过说话的方式，让另一个人在相同的位置也画出一个圆。

第六回合，先在两张白纸上分别画一个长方形，接着将长和宽平均分为多格，从0开始给长和宽的分割线标出数字，分配游戏角色。一个人在长方形中任意一个交叉点画一个圆，不能让另一个人看到。接着画的人需要通过说话的方式，让另一个人在相同的位置也画出一个圆。

想一想： 哪些回合成功在相同位置画出圆？每个回合用时分别是多少？用时少的回合为什么时间更短？将长方形替换为舞台，如何告诉别人角色在舞台上的位置？

# 【线索篇】

当你去向对方描述位置时，尝试着去设定一个参照物（尺子、手指），并通过符号将参照物标识出来用于交流（数字、文字）。

# 【解决篇】

一根直线，在上面画一个圆，如何用语言告诉别人这个圆在什么位置？

最开始，你可能想到的是，你们两知道哪些用于描述位置的词语。例如，上下左右，东南西北，它们都明确表示了某一个特定的位置。这时如果你在直线的最左边、最右边或者正中间画一个圆，就可以利用这种特定的词语告诉对方。

如果这个圆画在了特定词语之外的区域，应该怎么办？

这时你就需要和对方在直线上设置一种双方都能理解的参照物用于说明。例如，以手指的长度为单位，将直线分为几段，这时你就可以告诉对方自己的圆，画在了第几段上。

每个人手指的长度会有区别，为了避免误差，更准确的方法可以使用直尺，利用固定的长度单位对直线进行划分，这样对方就可以更精准的找到圆在直线上的位置。

有了参照物，对方已经可以准确的定位到圆在直线的位置，但是对你的描述来说还不是最方便的形式。例如，你想要向对方描述第二回合那个圆的位置时，你必须说它在左数或者右数第几根竖线上，对方听完以后还需要根据你的描述进行计算，在直线上找到对应的点位。

那最方便的形式是什么？就是你们利用符号为每一根竖线取一个名字，这样只要告诉对方竖线的名字，对方就能立即找到对应的位置。

最为常用的，就是在第三回合使用的数线坐标系open in new window。将直线的中间标记为0，取名叫原点。向右数字递增，向左数字递减。你可以把数字看做一个点到原点的距离，正负号则代表它到原点的方向。

有了数线坐标系，你就可以将直线上任意一点的数字告诉对方，对方就能在相同的位置画出圆。

接着，来到第四回合。后面三个回合看上去和之前很相似。开始是一个方形，接着利用竖线将方形分成了很多格，最后给每条线用数字进行标记。

它们的区别又是什么？

想象一下，第四回合的方框是你的房间，你就是那个圆圈，如果你想要告诉别人你的位置，应该怎么表达？你还能不能像前三回合一样，画一条线，告诉别人你在线上的位置？例如，你可以把自己站的位置与任意一个墙角连成一条线，再把这条线与墙角其中一个面的角度告诉对方，它就能画出和你一样的直线，找到圆圈的位置。只是这样会有一个问题，一旦你的位置发生了变化，你就必须重新测量角度画一条直线。

仔细观察，无论圆圈在方框中的什么位置，你都可以利用两个互相垂直的数线坐标系来表示它，这两个竖线坐标系相交于原点处，也叫做原点。其中一条线可以告诉你，圆圈到原点左右方向的距离(x)，另一条线可以告诉你圆圈到原点上下方向的距离(y)。这就是直角坐标系。通过它，你就可以利用两个数字(x, y)，表示出方框中任意一个圆圈的位置。

尝试着多做几轮第6回合的游戏，看看谁能描述的更准确，谁又能画的更快。

做完游戏后想一想，Scratch中的舞台是不是一个方框？角色是不是里面的圆圈？当你想告诉别人角色在舞台中的位置时，你应该如何做？

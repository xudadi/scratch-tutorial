# 小技巧

在Scratch中有一些小技巧可以帮助你快速完成一些任务，提高自己的开发效率。

# 修改坐标参数

在使用运动分类中的移动积木时，比较麻烦的一件事情就是需要手动去更改积木中的坐标参数，有一个简单的方法可以完成它。

每当你拖动舞台中的角色时，积木块中的坐标参数会随着角色位置的改变而改变。因此，在你将移动积木拖入编程区之前，可以先将角色拖到目标位置，这时就不再需要手动修改坐标参数值。需要注意，拖动角色时程序必须是停止状态。

# 复制指令

你在开发过程中经常会遇到的问题之一，就是需要使用一些重复指令串。复制指令串有三种方式。

# 复制一段指令

当你将鼠标放在一块积木上时，单击鼠标右键，会出现一个菜单栏。选中菜单中的复制，就可以得到单击积木块，以及它下面连着的所有积木块。

# 复制整段指令

如果你需要把一个角色的整段指令复制给另外一个角色，只需要拖动该指令串到另一个角色的头像处就可以了。

# 复制角色

你在一个作品中可能会用到多个相同的角色，它们的指令操作都是一样的，你只需要右键角色的头像，选择复制，就可以得到一个一样的角色。

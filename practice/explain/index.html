# 练习说明

学习的关键在于持续不断(数量)的有效练习(质量)，针对每个积木块的用法，我将会从易至难，给你提供不同的项目练习案例（陆续更新）。每个项目的结构还是分为 提出目标(疑惑篇)——>收集线索(线索篇)——>实现目标(解决篇) 三个部分，帮助你更好的学习Scratch，最终可以自己完成有趣的动画或游戏制作。

案例演示分为视频演示和亲自操作，亲自操作部分由于需要加载的文件较大，首次打开时间比较长。

在练习的过程中，还是以自主实现案例为主要目标。先观看案例演示(疑惑篇)，尝试自己按照练习步骤去完成它。如果完成不了，参看每个案例的线索提示(线索篇)，获得线索后继续尝试自己完成作品。当已经完成作品或者确实无法自己完成时，再去观看案例制作过程解析(解决篇)，与自己的思考过程做对比，总结指令的用法和制作规律。

案例制作过程的解析，我将会通过微信公众号”梦世界“持续推送，欢迎关注。

如果你在练习过程中希望获得针对自己所遇问题的反馈，欢迎加入由我主理的付费社群石头解忧杂货店向我提问。
